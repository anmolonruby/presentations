iD

is a pure-javascript, complete map editor for OpenStreetMap designed to be easy to learn and simple to use

we* wrote it in the past ~7 months at MapBox under a Knight News Grant

we*

John Firebaugh, Ansis Bramannis, Saman Bemel-Benrud, myself, and many volunteers

why

usability 111 + design 37

iD tries to be well-designed from day one.

Details like icons, custom cursors, and color schemes were made from scratch and developed inline with code changes.

The Goal

Replace Potlatch 2 as the default editor on OpenStreetMap for Internet Explorer 10+

Progress

v0.x: the basic skeleton

v1.0: presets, details, documentation, translations

v1.1: relations editing + performance

Performance

Good in Chrome, buggy in Opera, weak in Firefox

Low-hanging-performance fruit is mostly eaten

Browser vendors are helping! We have multiple bugs fixed & optimizations queued in WebKit + Gecko.

What's Next

  • The Default! We Hope.
  • Modularity
  • Plugins

Plugins*

Very important. There are no bounds on what people expect an editor to do. Chat, alignment fixes, WMS, conflation, kitchen sink, etc. If we implemented all requested features we would be 'done' (never)
